PROJECT OVERVIEW
════════════════════════════════════════════════════════════════════════════════

The Royal E-Cars WhatsApp Chatbot is a fully-implemented RAG (Retrieval-
Augmented Generation) system that intelligently answers customer questions
about luxury electric vehicles using semantic search and advanced LLMs.

Technology Stack:
  ✅ FastAPI (async web framework)
  ✅ OpenAI GPT-3.5-turbo (LLM)
  ✅ Chroma DB (vector database)
  ✅ Twilio WhatsApp API (messaging)
  ✅ Python 3.11+

SYSTEM ARCHITECTURE
════════════════════════════════════════════════════════════════════════════════

MESSAGE FLOW:
  WhatsApp Message
    ↓
  Twilio API Gateway
    ↓
  FastAPI Webhook (/whatsapp/webhook)
    ↓
  RAG PIPELINE:
    1. Extract message from payload
    2. Generate query embedding (OpenAI)
    3. Semantic search in vector DB (Chroma)
    4. Retrieve top-5 similar documents
    5. Build context prompt
    6. Call OpenAI GPT with context
    7. Generate response
    ↓
  Context-Aware Features:
    - Detect if booking-related
    - Inject booking link automatically
    ↓
  Send Response + Booking Link via Twilio
    ↓
  WhatsApp Reply to Customer

CONFIGURATION OPTIONS
════════════════════════════════════════════════════════════════════════════════

OPENAI:
  OPENAI_API_KEY - Your OpenAI API key (required)
  OPENAI_MODEL - Model name (default: gpt-3.5-turbo)

TWILIO:
  TWILIO_ACCOUNT_SID - Your Twilio account ID (required)
  TWILIO_AUTH_TOKEN - Your Twilio auth token (required)
  TWILIO_WHATSAPP_NUMBER - WhatsApp sender number

APPLICATION:
  ENVIRONMENT - dev/production mode
  CHROMA_PERSIST_DIR - Vector DB location

RAG PARAMETERS:
  TOP_K_RETRIEVAL - Number of documents to retrieve (default: 5)
  CHUNK_SIZE - Document chunk size (default: 500)
  CHUNK_OVERLAP - Chunk overlap for context (default: 50)
  EMBEDDING_MODEL - Embedding model name (default: text-embedding-3-small)


GETTING STARTED
════════════════════════════════════════════════════════════════════════════════

1. INSTALLATION
   cd C:\Users\arsen\Desktop\rag-app\royal_ecars_chatbot
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt

2. CONFIGURATION
   Copy .env.example to .env
   Fill in your credentials:
     - OPENAI_API_KEY
     - TWILIO_ACCOUNT_SID
     - TWILIO_AUTH_TOKEN

3. INITIALIZATION
   python setup.py

4. TESTING
   python scripts/test_rag_pipeline.py

5. RUNNING
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

6. WHATSAPP SETUP
   Set up Twilio webhook: POST https://your-url/whatsapp/webhook
   Send test message to WhatsApp number
   Receive intelligent response!

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

If you encounter issues:

1. Import Errors
   └─ Ensure you're in the correct directory
   └─ Verify virtual environment is activated

2. API Errors
   └─ Check credentials in .env
   └─ Verify API keys are valid
   └─ Check account has sufficient credits

3. Database Errors
   └─ Run: python database/seed_database.py
   └─ Delete chroma_db/ folder and reseed if corrupted

4. Webhook Not Working
   └─ Use ngrok: ngrok http 8000
   └─ Configure webhook URL in Twilio dashboard
   └─ Check webhook logs
