═══════════════════════════════════════════════════════════════════════════════

    User: "Wie viel kostet eine Tour?"
    
         │
         ↓
    
    Retrieve similar documents
    ├─ Topic: "Pricing" → doc_2 (booking info)
    ├─ Relevance: 0.92 (high)
    │

       "Klassische Wien Tour (2h): €150"
       "Gourmet Tour: €280"

         │
         ↓
    
    Generate context prompt:
    
    ┌─────────────────────────────────────────┐
    │ System: Du bist ein Kundenservice...    │

6. ERROR HANDLING FLOW
═══════════════════════════════════════════════════════════════════════════════

    ┌─────────────────┐
    │ Incoming Message│
    └────────┬────────┘
             │
             ↓
    Try to process
    │
    ├─ Error: OpenAI API key missing?
    │  └─ Fallback: "Entschuldigung, Systemmehler"
    │
    ├─ Error: Vector DB not initialized?
    │  └─ Fallback: "Datenbank nicht erreichbar"
    │
    ├─ Error: Twilio credentials invalid?
    │  └─ Fallback: "Nachricht konnte nicht gesendet werden"
    │
    ├─ Error: Embedding generation failed?
    │  └─ Fallback: Dummy embedding or skip retrieval
    │
    ├─ Error: LLM generation timeout?
    │  └─ Fallback: Return error message
    │
    └─ Success: Send full response
       └─ Log to console
